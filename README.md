不同字體列印的《黃自元．間架結構 92 法》。

有些字體字數較少，印製字帖時會有缺字狀況，遇到這個情況時系統會以『隨峰體 Plus』填補。因此，若您發現怎麼字體造型忽然變了，那就是原字體缺字，以隨峰體補上了。

像『辰宇落雁體』是兩個高中生的自主學習主題，受限於課業時間，他們的字體就有一些缺字，所以用其他字體補足，避免留空。
